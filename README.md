# 🚗 Car Price Predictor - Frontend

This project is the **frontend interface** for a Car Price Prediction system built using **Streamlit**.  
The application allows users to input car details and receive a predicted car price from the backend prediction API.

---

## 📌 Project Overview

This application provides a simple user interface where users can:

- Select car details such as name, company, year, fuel type, and kilometers driven
- Send the input data to a prediction API
- Display the predicted car price returned by the backend model

The frontend is built using **Streamlit**, which allows quick development of interactive data applications in Python.

---

## ⚙️ Technologies Used

- Python
- Streamlit
- Pandas
- Requests

---
## 💻 Features

- Interactive user interface
- Dropdown selection for car details
- API integration for predictions

---

## 📊 Input Features

The application collects the following inputs:

- Car Name
- Company
- Year
- Fuel Type
- Kilometers Driven

These inputs are sent to the backend prediction API to generate the predicted car price.

---

## 🔗 Backend API

The frontend communicates with a backend prediction API that performs the machine learning inference and returns the predicted price.
